日程添加部分

1 智能添加

除了概述部分之外，智能添加页需要注意以下几点：

  1. 优先语音添加
  2. 语音识别错误时，用户可以很方便地修改
  3. 便于在语音添加和文字添加之间切换
  4. 快速的切换日程类别
  5. 快速切换添加的是“日程”还是“待办”
  6. 非常方便地选择时间
    • 日期选择控件可以切换全天
    • 日期选择控件可以切换农历和公历
    • 日期选择控件可以快速切换年
    • 日期选择控件可以快速切换月
    • 时间选择控件：如果跨天了需要提示用户
    • 时间选择控件：显示其他日程，以防时间冲突
    • 可以切换为传统的时间选择方式

2 完整添加

智能添加点击完成之后，进入完整添加页面

完整添加页面优先提示用户关联人

完整添加页面提供以下部分

  1. 快捷方便地关联人
    • 应用内好友
    • 推荐的好友
    • 常用的好友
  2. 快捷的添加地点，并能导航
  3. 方便地分享到微信
  4. 日程详情可以添加图片和附件
