添加日程需求文档

1 概述

1.2版添加日程的原则是：简单

删除了原来的完整添加，将管理员的日程详情页面和完整添加整合

添加页面（页面名称是智能添加）只提供主题和时间

2 智能添加

点击添加页面，进入“智能添加页面”，如下图所示

2.1 语音添加

按住语音按钮，即可语音添加日程

语音输入时，按钮为水波纹效果。待输入完成，则显示“正在识别”

识别成功后，直接显示“智能添加页面”

2.2 文字添加

点击输入框，则采用文字添加

在用户输入文字的过程中，自动识别时间，在确定输入内容已经不是时间的时候，将代表时间的文字添加背景

2.3 编辑时间

不论是语音添加还是文字添加，都会出现时间识别错误的情况

点击时间区域，即可修改时间

点击时间区域，屏幕内出现蒙版。时间选择框从上向下出现。之后再向右滑出；与此同时日期选择框从左向右滑出

2.4 编辑类别

2.4.1 自动编辑类别

识别关键词，自动编辑类别

  1. 工作：开会、讨论、会议
  2. 娱乐：聚餐、聚会、看电影、生日
  3. 学习：看书，上课，复习，自习，作业，交流

没有关键词，则根据用户输入的时间，自动编辑类别：

  1. 工作:默认
  2. 娱乐:周末的事件

2.4.2 手动编辑类别

点击主题旁边的分类标志，选择分类

确定分类，输入框颜色变化

手动选择分类之后，不再自动识别分类

3 日程详情（可编辑）

点击智能添加页面的“保存”，进入创建者的日程详情页面（也可以理解为完整添加页面，但是此页面提供转发到微信功能）。

默认提醒时间为提前1小时，并可以在设置中修改

3.1 编辑时间

编辑时间分为编辑日期和编辑小时

点击日期，弹出日期编辑弹框，在日期弹框中选择是否为全天日程；一旦选择全天日程，则不再能够左滑进入编辑时间页面

点击时间，进入编辑起始时间空间。长按日程部分可以拖动整体时间区域；单击圆圈则可修改时间。日程颜色框内实时显示起始时间和时长

3.2 编辑地点

点击地点按钮，地点输入框滑动到屏幕上方。输入地点时识别地点并推荐

用户点击推荐地点之后，推荐栏消失，出现地图

点击保存，回到日程详情页面

3.3 编辑关联人

3.4 分享到微信

可编辑的日程详情页面，右上角为分享按钮，点击分享，弹出“微信”和“朋友圈”的选择框
